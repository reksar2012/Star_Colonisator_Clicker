Local History for Unity3d automatically creates a history of your files every time they are saved. 
You can revert prefab version to any time when its modified.

How to test plugin:
- Modify and save scene 3 times. For example create cube then move it arround while saving scene.
- Go to Window/Local History
- Select current scene in project folder. In local history window appears file history.

backup files stored in "project folder/History"
